# Janespot - Portfolio

Personal portfolio website for Jane Odhiambo, AI Engineer & Full-stack Developer.

**Live:** [janespot.github.io](https://janespot.github.io)

## Tech Stack

- React 18 + Vite
- SCSS with dark/light theme
- Formspree contact form
- GitHub Actions for deployment

## Sections

Hero · About · Skills · Experience · Education · Portfolio · Certifications · Volunteer · Contact

## Getting Started

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

Deploys automatically to GitHub Pages on push to `main` via GitHub Actions.
